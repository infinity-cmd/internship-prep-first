## 🎯 项目1：词频统计（1_词频统计）
### 核心功能
基于 Python 解析《八十天环游地球》小说文本，按章节拆分内容，利用分词工具提取每章高频词汇并统计出现次数，生成结构化统计结果。

### 关键文件说明
| 文件名                | 作用                     |
|-----------------------|--------------------------|
| statistics.py         | 核心脚本（分词、统计、结果输出） |
| 八十天环游地球.txt    | 小说源文本（完整内容）|
| 八十天环游地球-章节.txt | 拆分后的章节文本（自动生成） |
| 章节高频词统计.txt    | 高频词统计结果（自动生成） |

### 运行方式

```bash
# 安装依赖（如需分词功能）
pip install jieba

# 执行词频统计脚本
python statistics.py

```
